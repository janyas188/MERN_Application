# MERN Application

Webpage application created using MongoDB, ExpressJS, ReactJS, and NodeJS.

## Installation

Use the package.json file and npm to install dependencies.

```bash
npm install
```
Change the **URL variable** in the  **Mongo_client.js**  to the URL of a local MongoDB database.

The database in application in the server had a collection which stored the value exactly as given in the **input.json** file. 

## Usage

Run app.js using 

```run from bash
npm start app.js
```

The server will run on port 3000

Access the Products page using /products

Access the Login page using /login



#### This application was created using Visual Studio 2017
